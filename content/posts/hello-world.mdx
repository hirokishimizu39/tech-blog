## はじめに

このブログへようこそ！これは **Next.js** と **MDX** を使って作成した技術ブログです。

## このブログの特徴

- **MDX** で記事を書ける
- **ダークモード** 対応
- **シンタックスハイライト** 付きのコードブロック
- **Vercel** で無料ホスティング

## コードの書き方

MDXではコードブロックを簡単に書けます：

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```
